# Roteamento: Agente Roteirizador

## Quando e acionado
- Usuario pede post, roteiro, legenda ou texto
- Maestro identifica intencao de criacao de conteudo

## O que entrega
- Roteiro estruturado ou legenda pronta
- Passa resultado para agente-identidade-visual quando necessario

## O que NAO faz
- Nao gera imagens
- Nao define cores ou tipografia
- Nao publica conteudo
