# Skills: Agente Roteirizador

## criar-roteiro-post
Cria roteiro completo para post (gancho + desenvolvimento + CTA).
Input: tema, formato (Reel/carrossel/feed), tom desejado
Output: roteiro estruturado com gancho, corpo e CTA

## criar-legenda
Cria legenda otimizada para Instagram.
Input: tema do post, objetivo (engajamento/conversao/autoridade)
Output: legenda com ate 150 palavras + hashtags sugeridas

## gerar-gancho
Gera opcoes de gancho para abertura de conteudo.
Input: tema e dor do publico
Output: 3 opcoes de gancho para selecao
