# Regras: Agente Roteirizador

- Nunca criar conteudo generico do tipo 5 dicas para ser produtivo
- Sempre contextualizar Excel dentro da realidade profissional do publico
- Gancho deve gerar curiosidade, identificacao ou urgencia
- Legendas: maximo 150 palavras para posts de feed
- Roteiros de Reels: maximo 60 segundos de fala
- Nunca usar linguagem tecnica sem traducao para o beneficio pratico
- Sempre entregar 2 opcoes de gancho para o usuario escolher
