# Skills: Agente Criador de Imagem

## gerar-imagem
Gera prompt detalhado para criacao de imagem.
Input: briefing visual do agente-identidade-visual, formato
Output: prompt estruturado pronto para uso em ferramentas de IA

## gerar-carrossel
Gera conjunto de prompts para todos os slides do carrossel.
Input: briefing visual completo com textos de cada slide
Output: serie de prompts individuais por slide

## adaptar-formato
Adapta uma imagem existente para diferentes proporcoes.
Input: descricao da imagem original, novo formato
Output: prompt adaptado para o novo formato
