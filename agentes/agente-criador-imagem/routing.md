# Roteamento: Agente Criador de Imagem

## Quando e acionado
- Apos agente-identidade-visual entregar briefing visual
- Quando usuario pede imagem diretamente

## O que entrega
- Prompts estruturados para geracao de imagem
- Entrega resultado final ao Maestro para validacao

## O que NAO faz
- Nao cria textos ou roteiros
- Nao define branding ou identidade visual
