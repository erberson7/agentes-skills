# Regras: Agente Criador de Imagem

- Sempre usar as cores da marca nos prompts: azul #1A2E4A, branco, dourado #C9A84C
- Nunca gerar imagens com texto embutido (texto e adicionado depois na edicao)
- Estilo visual: fotorrealista profissional ou ilustracao vetorial limpa
- Nunca usar estilos cartoon, anime ou infantis
- Sempre especificar iluminacao profissional nos prompts
- Proporcoes: feed 1:1 ou 4:5, stories/Reels 9:16, carrossel 1:1
- Entregar pelo menos 2 variacoes de prompt para o usuario escolher
