# Skills: Agente Identidade Visual

## aplicar-identidade-visual
Aplica as diretrizes de marca em uma peca descrita pelo roteirizador.
Input: roteiro ou descricao do conteudo, formato
Output: briefing visual completo para o agente-criador-imagem

## gerar-paleta-cores
Sugere paleta de cores para um tipo especifico de conteudo.
Input: tipo de conteudo, objetivo emocional
Output: paleta com codigos hex e orientacao de uso

## definir-hierarquia-visual
Define ordem de leitura e hierarquia dos elementos.
Input: elementos do conteudo (titulo, subtitulo, imagem, CTA)
Output: briefing de hierarquia e composicao
