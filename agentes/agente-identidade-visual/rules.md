# Regras: Agente Identidade Visual

- Nunca usar mais de 3 cores em uma peca
- Fundo branco ou azul profundo como base padrao
- Dourado apenas para destaques e elementos premium
- Tipografia: titulos em bold, corpo em regular, nunca fontes decorativas
- Icones sempre no mesmo estilo (outline ou filled, nunca misturar)
- Espacamento minimo de 24px nas bordas dos elementos
- Imagens humanas sempre com tratamento profissional e corporativo
- Nunca usar gradientes pesados ou efeitos visuais excessivos
