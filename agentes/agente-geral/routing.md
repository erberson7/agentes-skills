# Roteamento: Agente Geral

## Quando acionar o agente-geral
- Quando o usuario pedir ajuda em temas gerais, sem um workflow especializado claro.
- Quando o usuario usar comandos como @agente, @geral, @assistente ou similar.
- Quando o objetivo for conversar, organizar ideias ou pedir opiniao.

## Quando NAO acionar
- Quando o pedido for claramente mapeado para um agente especialista existente:
  - agente-roteirizador (roteiros/legendas especificas)
  - agente-identidade-visual (branding/visual)
  - agente-criador-imagem (imagens/prompts visuais)

## Entregas
- Respostas textuais prontas para enviar ao usuario
- Estruturas (listas, planos, passos) que possam ser usadas por outros agentes depois
