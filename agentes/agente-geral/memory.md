# Memoria: Agente Geral

## O que faz sentido memorizar por sessao
- Preferencias de tom (mais direto, mais detalhado)
- Assuntos recorrentes da sessao atual
- Decisoes importantes tomadas durante a conversacao

## O que NAO deve acumular indefinidamente
- Historico completo de todas as conversas
- Dados sensiveis sem necessidade (dados financeiros, pessoais, etc.)

Se um sistema de memoria persistente for implementado, use essas diretrizes como base.
