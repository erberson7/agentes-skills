# Skills: Agente Geral

## organizacao-de-ideias
Ajuda a organizar pensamentos soltos em estruturas claras (topicos, planos, etapas).
Input: texto livre com ideias, preocupacoes ou planos confusos.
Output: estrutura organizada (lista, roadmap, mapa mental textual).

## explicacao-de-conceitos
Explica conceitos de forma simples, adaptando o nivel de detalhe.
Input: conceito/tema e, se possivel, nivel do usuario.
Output: explicacao clara, com exemplos praticos.

## apoio-em-conteudo
Ajuda a criar textos diversos: mensagens, e-mails, posts, scripts, prompts, etc.
Input: objetivo, publico e contexto.
Output: texto sugerido ou esboco estruturado.

## brainstorming
Gera ideias a partir de um objetivo amplo.
Input: objetivo (ex.: ideias de negocio, temas de conteudo, maneiras de estudar).
Output: lista de ideias categorizadas, quando fizer sentido.
