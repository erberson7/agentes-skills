# Workflow: Gerar Legenda

## Objetivo
Criar legenda otimizada para um post ja existente ou em producao.

## Agentes envolvidos
1. agente-roteirizador

## Fluxo detalhado

### Etapa 1 - Input
- Usuario descreve o tema ou passa o roteiro do post
- Diretor Geral confirma objetivo (engajamento, conversao, autoridade)

### Etapa 2 - Criacao
- agente-roteirizador cria legenda com:
- Gancho na primeira linha
- Desenvolvimento em ate 100 palavras
- CTA claro
- Hashtags sugeridas (5 a 10)

### Etapa 3 - Entrega
- Legenda entregue ao usuario para validacao
- Ajustes se necessario
