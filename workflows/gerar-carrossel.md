# Workflow: Gerar Carrossel

## Objetivo
Criar um carrossel completo para o Instagram com textos e visual de cada slide.

## Agentes envolvidos
1. agente-roteirizador
2. agente-identidade-visual
3. agente-criador-imagem

## Fluxo detalhado

### Etapa 1 - Definicao
- Usuario define tema e numero de slides (recomendado: 5 a 8 slides)
- Diretor Geral confirma objetivo (educativo, conversao, autoridade)

### Etapa 2 - Textos dos slides
- agente-roteirizador cria texto para cada slide
- Slide 1: gancho | Slides intermediarios: desenvolvimento | Ultimo slide: CTA
- Usuario valida textos

### Etapa 3 - Visual
- agente-identidade-visual define estilo visual de cada slide
- agente-criador-imagem gera prompt para cada slide

### Etapa 4 - Entrega
- Diretor Geral entrega pacote completo: textos + briefings visuais + prompts
