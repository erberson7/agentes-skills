# Workflow: Criar Post Instagram

## Objetivo
Produzir um post completo para o Instagram da marca, desde o roteiro ate a imagem final.

## Agentes envolvidos
1. agente-roteirizador
2. agente-identidade-visual
3. agente-criador-imagem

## Fluxo detalhado

### Etapa 1 - Briefing inicial
- Usuario instrui o Diretor Geral via WhatsApp
- Diretor Geral confirma: tema, formato, objetivo do post

### Etapa 2 - Roteiro
- agente-roteirizador cria gancho + corpo + CTA
- Entrega 2 opcoes de gancho para o usuario escolher
- Usuario valida ou pede ajuste

### Etapa 3 - Identidade Visual
- agente-identidade-visual recebe roteiro aprovado
- Gera briefing visual completo (cores, tipografia, composicao)

### Etapa 4 - Criacao de Imagem
- agente-criador-imagem recebe briefing visual
- Gera 2 opcoes de prompt para geracao de imagem
- Usuario escolhe e imagem e gerada

### Etapa 5 - Entrega
- Diretor Geral compila: texto + legenda + imagem
- Envia para aprovacao final do usuario
