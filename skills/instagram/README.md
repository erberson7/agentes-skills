# Skills: Instagram

Conjunto de skills voltadas para criacao e publicacao de conteudo no Instagram.

## Skills disponiveis
- criar-roteiro-post: roteiro completo para post
- criar-legenda: legenda otimizada com hashtags
- gerar-gancho: opcoes de gancho para abertura
- gerar-carrossel: textos e visual para carrossel
- adaptar-formato: adaptar conteudo para feed, stories ou Reels

## Formatos suportados
- Feed: 1:1 e 4:5
- Stories: 9:16
- Reels: 9:16
- Carrossel: 1:1 por slide
