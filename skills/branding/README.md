# Skills: Branding

Conjunto de skills voltadas para manutencao e evolucao da identidade de marca.

## Skills disponiveis
- aplicar-identidade-visual: aplicar branding em qualquer peca
- gerar-paleta-cores: sugerir paleta para ocasioes especificas
- definir-tom-comunicacao: definir tom para tipo de conteudo
- criar-guia-marca: documentar diretrizes da marca
