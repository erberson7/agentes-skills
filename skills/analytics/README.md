# Skills: Analytics

Conjunto de skills voltadas para analise de metricas e performance de conteudo.

## Skills disponiveis
- analisar-metricas-instagram: leitura de alcance, engajamento e conversao
- comparar-performance: comparar posts por formato e tema
- sugerir-otimizacoes: sugerir melhorias baseadas em dados

## Fontes de dados
- Instagram Insights
- Planilhas de controle manual
- Relatorios exportados
