# Skills: Pesquisa

Conjunto de skills voltadas para pesquisa de referencias, tendencias e conteudo.

## Skills disponiveis
- pesquisar-tendencias-instagram: buscar tendencias do nicho no Instagram
- pesquisar-referencias-visuais: buscar referencias visuais para briefings
- pesquisar-pauta: buscar pautas relevantes para o publico da marca
- analisar-concorrencia: analise de posicionamento de concorrentes

## Fontes recomendadas
- Instagram (busca nativa e ferramentas de analytics)
- Google Trends
- LinkedIn para tendencias corporativas
