# Skills: Automacao

Conjunto de skills voltadas para automacao de processos operacionais.

## Skills disponiveis
- criar-workflow-n8n: criar workflow de automacao no n8n
- integrar-whatsapp: configurar integracao com WhatsApp via Evolution API
- agendar-publicacao: agendar posts para publicacao automatica
- notificar-usuario: enviar notificacoes ao usuario por WhatsApp

## Stack de automacao
- n8n: orquestracao de workflows
- Evolution API: integracao WhatsApp
- GitHub: repositorio de agentes e skills
